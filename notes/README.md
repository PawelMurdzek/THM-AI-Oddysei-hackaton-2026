# Notes

Cross-cutting notes, cheatsheets, and session summaries that don't belong to a single writeup.

## What goes here

- **Tool configs / cheatsheets** — e.g. `garak-config.md`, `burp-collaborator-notes.md`
- **Reusable payload lists** — jailbreak primitives, system-prompt extractors
- **Session summaries** — short logs after a long session, so we don't lose context
- **Reference material** — links, papers, related research

## What does *not* go here

- Per-challenge writeups → `writeups/NN-slug/README.md`
- Screenshots tied to a specific challenge → `writeups/NN-slug/assets/`
- Secrets, flags, credentials → never commit

## Naming

Free-form. Keep filenames descriptive: `jailbreak-payloads.md`, `2026-05-14-session.md`, `rag-poisoning-references.md`.
